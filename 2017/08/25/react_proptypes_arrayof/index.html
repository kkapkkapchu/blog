React 에서 속성의 타입을 정의할 때 propTypes를 사용하면 편하다.
속성의 타입이 array 라면 배열 안의 값에 대한 타입도 arrayOf()로 지정할 수 있다.

아래 예시는 배열 안의 값의 타입을 숫자로 정의한 것이다.
